## Animal Information Generator

## Project Description

This project does amazing things and solves all your problems!

This project is a Python-based animal information generator that retrieves animal data from an external API and dynamically creates an HTML webpage. Users can enter the name of an animal, and the application fetches details such as its diet, location, and type before inserting the information into a predefined HTML template. The generated webpage provides an easy-to-read summary of the animal and is automatically saved as an HTML file for viewing in a web browser.

### Features

* Fetches animal information from an external data source.
* Generates dynamic HTML content using a template.
* Displays key animal characteristics, including diet, location, and type.
* Handles invalid animal names gracefully with a user-friendly message.
* Automatically creates a ready-to-view HTML webpage.


## Installation

To install this project, simply clone the repository and install the dependencies in requirements.txt using `pip`

## Usage

To use this project, run the following command - `python main.py`.
